# Purdue Comminity App

Purdue Comminity App is a forum for Purdue Students to discuss all kinds of topics.

# Current Stage
## What has been done:
  * Forum page frontend,
  * Setting page frontend(barebone)
## What is not done:

  * Post detail page
  * Home page
